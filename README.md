# Experiment 42: Student Class (Constructors)

## Problem Statement

Create a `Student` class with attributes `name` (String) and `rollNo` (int).
Include a parameterized constructor to initialize these attributes.
Write a main method to create two `Student` objects with data read from input and display them.

## Input Format

* Line 1: Name of student 1.
* Line 2: Roll number of student 1.
* Line 3: Name of student 2.
* Line 4: Roll number of student 2.

## Output Format

* Student 1 details: `Name: [Name] Roll: [RollNo]`
* Student 2 details: `Name: [Name] Roll: [RollNo]`

### Example 1

**Input:**

```text
Alice
101
Bob
102
```

**Output:**

```text
Name: Alice Roll: 101
Name: Bob Roll: 102
```
